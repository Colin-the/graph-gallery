    Graph Gallery Help

    Datasets

    • MIMIC-III — Medical Information Mart for Intensive Care III; the primary dataset for both pipelines and all comparison experiments.
    • MIMIC-IV — Updated MIMIC with more recent data (EHR-Dataset-Processing pipeline only).
    • eICU — eICU Collaborative Research Database (EHR-Dataset-Processing pipeline only).
    • Cross — Pipeline comparison graphs that span both MIMIC_Extract and EHR-Dataset-Processing.

    Pipelines

    • MIMIC_Extract — Published Wang et al. 2020 pipeline, ported to Python 3.9 / Apptainer. Outputs 13 cohort scenario variants from a single Postgres MIMIC-III instance.
    • EHR-Dataset-Processing — In-house pipeline supporting MIMIC-III, MIMIC-IV, and eICU. Applies 12 sequential filters (7 per-vital outlier + 4 structural) then 5 aggregation methods; uses cuML RandomForest on GPU.
    • Comparison — Cross-pipeline notebook (pipeline_comparison.ipynb) comparing the two on a shared 27,295-stay MIMIC-III cohort with logistic regression / McNemar tests.

    Aggregation Methods

    Both pipelines support the same five methods for collapsing the 24-hour timeseries into a feature vector:

    • Mean — average value per vital across all hours
    • Median — median value per vital
    • Standard Deviation — spread of values per vital
    • Mean Deviation — mean absolute deviation from the mean
    • Maximum Deviation — maximum absolute deviation

    Labels

    • icu / icu_los — ICU length of stay > 3 days (binary)
    • mortality — in-hospital mortality (binary)

    Plot Types

    • Filter Impact Bar — deviation from raw (no-filter) baseline for a given metric; green = improvement, red = degradation.
    • McNemar Significance — −log₁₀(p-value) for each filter/scenario vs baseline; orange bars cross the p=0.05 threshold line.
    • Centroid Shift — per-vital deviation of class centroid after filtering; shows which vitals are affected most.
    • Centroid Distribution — density violin plots for raw vs filtered positive/negative groups per vital.
    • Lens Plot — aggregation method × hour heatmap for a single vital in the MIMIC_Extract baseline cohort.
    • PCA Centroid Shift — 2-D PCA scatter showing baseline (red star) vs variant scenario centroids with connector lines.
    • Error-Bar — mean ± 1 std dev + median across scenarios for a single vital.
